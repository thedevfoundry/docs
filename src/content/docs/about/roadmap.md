---
title: Roadmap
description: A simple, evolving roadmap for Dev Foundry growth.
---

This roadmap is intentionally lightweight. We update it as the community learns.

## Phase 1: Foundation (now)

- Launch core docs and contribution flow
- Establish Discord channels and participation rhythm
- Run consistent coworking sessions
- Start monthly demo nights with repeatable format

## Phase 2: Momentum (next)

- Improve onboarding for new members
- Increase weekly progress reporting consistency
- Grow cross-functional collaboration on projects
- Publish reusable templates for demos and updates

## Phase 3: Incubation (later)

- Support team formation around validated ideas
- Create lightweight founder support workflows
- Track launches, experiments, and outcomes
- Expand local visibility of shipped community projects

## Success metrics

We evaluate roadmap progress through:

- number of active builders shipping each month
- number of demos and project updates shared
- number of collaborations formed through the community
- number of products, launches, or startups emerging

## How roadmap changes

Anyone can propose roadmap updates through GitHub issues or pull requests. Changes should be based on observed community behavior and clear impact on builder outcomes.
